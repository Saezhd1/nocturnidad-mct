# Nocturnidad TITSA – Subida múltiple y exportación a PDF

## Funcionalidad
- Subir múltiples PDFs del modelo "Relación de datos variables por trabajador".
- Extraer HI/HF por fecha, calcular minutos nocturnos en franjas 22:00–00:59 y 04:00–06:00.
- Aplicar tarifas: 0,05 €/min (30/04/2022–25/04/2025) y 0,062 €/min desde 26/04/2025.
- Mostrar resultados por día, resumen mensual y global.
- Descargar PDF con nombre completo y número de empleado incrustados.
